HaruFit - 요구사항 명세서

1. 프로젝트 개요

1.1 프로젝트명

HaruFit - 개인 맞춤형 운동 관리 웹 애플리케이션

1.2 프로젝트 목적

사용자가 자신의 운동 루틴을 관리하고, 운동 기록을 추적하며, 목표를 설정하고 달성할 수 있는 종합적인 피트니스 관리 플랫폼을 제공합니다.

1.3 프로젝트 범위

  • 사용자 인증 및 프로필 관리
  • 운동 루틴 생성 및 관리
  • 운동 기록 및 통계
  • 목표 설정 및 추적
  • 운동 일지 작성

2. 기능 요구사항

2.1 사용자 관리

기능 설명 우선순위
회원가입 이메일, 비밀번호, 닉네임으로 회원가입 높음
로그인/로그아웃 세션 기반 인증 시스템 높음
프로필 관리 신체 정보(키, 몸무게, 나이 등) 입력 및 수정 중간

2.2 운동 루틴 관리

기능 설명 우선순위
루틴 생성 운동 루틴 이름, 설명, 운동 종목 추가 높음
루틴 수정/삭제 기존 루틴의 운동 종목, 세트, 횟수 수정 및 삭제 높음
루틴 목록 조회 사용자가 만든 모든 루틴 목록 표시 높음
루틴 실행 루틴 선택 후 운동 시작, 세트별 기록 입력 높음

2.3 운동 기록 관리

기능 설명 우선순위
운동 기록 저장 운동 완료 후 세트별 무게, 횟수, 시간 기록 높음
기록 조회 날짜별, 운동별 기록 조회 높음
통계 및 차트 운동량, 무게 증가 추이, 운동 빈도 등 통계 시각화 중간

2.4 목표 관리

기능 설명 우선순위
목표 설정 체중, 근육량, 운동 빈도 등 목표 설정 중간
목표 진행률 목표 달성 진행률 표시 중간
목표 수정 기존 목표 수정 및 삭제 낮음

2.5 운동 일지

기능 설명 우선순위
일지 작성 운동 후 느낀 점, 메모 작성 중간
일지 조회 날짜별 일지 조회 중간
일지 수정/삭제 기존 일지 수정 및 삭제 낮음

3. 비기능 요구사항

  • 성능: 페이지 로딩 시간 3초 이내
  • 보안: 비밀번호 암호화 저장, SQL Injection 방지
  • 호환성: 최신 브라우저 지원 (Chrome, Safari, Firefox, Edge)
  • 반응형: 모바일, 태블릿, 데스크톱 화면 지원
  • 사용성: 직관적인 UI/UX, 쉬운 네비게이션

4. 데이터베이스 설계

4.1 주요 테이블

테이블명 설명 주요 필드
users 사용자 정보 user_id, email, password, nickname, height, weight, age, created_at
routines 운동 루틴 routine_id, user_id, name, description, created_at
exercises 운동 종목 exercise_id, name, category, muscle_group
routine_exercises 루틴별 운동 routine_exercise_id, routine_id, exercise_id, sets, reps, weight, rest_time
workout_logs 운동 기록 log_id, user_id, routine_id, workout_date, duration, notes
workout_sets 세트별 기록 set_id, log_id, exercise_id, set_number, weight, reps, rest_time
goals 목표 goal_id, user_id, goal_type, target_value, current_value, deadline, status
workout_journals 운동 일지 journal_id, user_id, workout_date, content, created_at

5. 기술 스택

5.1 프론트엔드

  • HTML5, CSS3
  • Bootstrap 5 (반응형 UI)
  • JavaScript (Vanilla JS)
  • Chart.js 또는 유사 라이브러리 (통계 차트)

5.2 백엔드

  • PHP 8.0+
  • MySQL/MariaDB
  • PDO (데이터베이스 접근)

5.3 인프라

  • Docker & Docker Compose
  • Nginx (웹 서버)
  • PHP-FPM

6. 화면 구성

6.1 주요 페이지

  • 프로필 관리 페이지
  • 루틴 목록 페이지
  • 루틴 생성/수정 페이지
  • 운동 실행 페이지
  • 운동 기록 조회 페이지
  • 통계 및 차트 페이지
  • 목표 관리 페이지
  • 운동 일지 페이지
